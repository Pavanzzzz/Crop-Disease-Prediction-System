# Crop Disease Prediction System

A Deep Learning-based Crop Disease Prediction System that identifies crop diseases from leaf images using TensorFlow/Keras and provides predictions through a Flask web application.

## Features

* Upload crop leaf images
* Predict plant diseases using a trained deep learning model
* User-friendly web interface
* TensorFlow/Keras-based classification model
* Fast and accurate disease detection

## Technologies Used

* Python
* Flask
* TensorFlow
* Keras
* NumPy
* OpenCV
* HTML/CSS
* Jupyter Notebook

## Installation

### Clone Repository

```bash
git clone https://github.com/Pavanzzzz/Crop-Disease-Prediction-System.git
cd Crop-Disease-Prediction-System
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```
