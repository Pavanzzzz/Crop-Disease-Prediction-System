# Crop Disease Prediction System

A deep learning-based crop disease prediction system built using TensorFlow and Flask.

## Note
The dataset and trained model files are not included in this repository due to their large size.

## Features
- Crop disease classification
- Flask web application
- TensorFlow/Keras model
- Image upload and prediction

## Technologies Used
- Python
- TensorFlow
- Keras
- Flask
- OpenCV
